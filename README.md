# AKPsi App

An internal web app for the Alpha Kappa Psi (AKPsi) chapter at the University of Washington. Built to help members, officers, and pledges stay connected and organized.

## Features

- **Announcements** — Post and view chapter announcements
- **Member Registry** — Browse member profiles with contact info, major, pledge class, and more
- **Family Tree** — Visualize big/little relationships across pledge classes *(coming soon)*

## Tech Stack

- Next.js (App Router), React, TypeScript
- MUI + Tailwind CSS
- Firebase (Auth, Firestore)

## Getting Started

```bash
npm install
cp .env.local.example .env.local  # add your Firebase config
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
